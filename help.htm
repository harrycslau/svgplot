Help for SVGplot 1.1 beta

Created by Harry Lau, 2010-2015

This program use SVG, a vector-based image format, to plot textbook-like graphs. The output is automatically converted to PNG, a common image format, for easy manipulation.

A. Basic Tags
There are 6 basic tags which must be included in each plot.

xlabel[offset]=text
Define label for the x-axis.
Offset could be specified, taking right as positive. Example:
xlabel=Time \it{t} / s
The label looks like: Time t / s

xparam=x1,x2,int,skip
The x-axis starts from x1 to x2, with an interval of int.
You may also skip some label number.
Examples:
xparam=0,60,10
The x-axis goes like 0,10,20,30,40,50,60.
xparam=0,60,20
The x-axis goes like 0,20,40,60.
xparam=0,60,10,1
The x-axis goes like 0,20,40,60, where the space of 10,30,50 are still here.

ylabel=text
Define label for the y-axis.
Example:
ylabel=Volume / cm^{3}
The label looks like: Volume / cm3

yparam=y1,y2,int,skip
The y-axis starts from y1 to y2, with an interval of int.
For the example of skip, see xparam.
Example:
yparam=-4,6,2
The y-axis goes like -4,-2,0,2,4,6.

subdiv=n
How many sub-divisions between one division.
Usual values are 1, 2, 5 and 10.
Example:
subdiv=5
There are 5 sub-divisions between one division.

gridsize=n
Size of the graph.
Default value is 1. Decimals (e.g. 1.5) are allowed.
Example:
gridsize=2
The size of the graph is doubled.


B. Function Tags

line[color]=x1,y1;x2,y2;x3,y3;...
Plot straight lines to connect between two points.
Default color is black, which is optional.
Example:
line=0,0;20,100;
line[red]=0,100;50,40;10,80;

dash[color]=x1,y1;x2,y2;x3,y3;...
Plot straight dashed lines to connect between two points.
Default color is black, which is optional.
Example:
dash=0,0;20,100;
dash[#00FF00]=0,100;50,40;10,80;

curve[color]=x1,y1;x2,y2;x3,y3;...
Plot curved lines to connect the points.
Default color is black, which is optional.
Example:
curve[blue]=0,0;2,4;3,9;4,16;

points[color]=x1,y1;x2,y2;x3,y3;...
Indicate the points given with a cross.
Default color is black, which is optional.
Example:
points[blue]=0,0;2,4;3,9;4,16;

mark[color,off-x,off-y]=x,y,text
Mark down the text on the graph at coordinates (x,y).
Default color is black, which is optional.
off-x and off-y are offsets in pixels, where +ve offset means moving right and down. Defaults are 0, which is optional.
Example:
mark=10,15,Hello
mark[red]=0,2,\it{N}_{0}
mark[orange,0,-10]=4,7,John

equation[color]=equation
(Experimental) Plot the given equation according to the range of x.
The equation must be a function of x (cannot be y, t, etc.)
Only functions that are continuous within the range of x can be plotted.
Default color is black, which is optional.
Supported functions are listed below:
'sqrt','abs','ln','log',
'sin','sinh','arcsin','asin','arcsinh','asinh',
'cos','cosh','arccos','acos','arccosh','acosh',
'tan','tanh','arctan','atan','arctanh','atanh'.
Example:
equation=3*x+1
equation[blue]=e^(-0.5x)
equation=ln(0.0001+x)
equation[red]=sin(x*pi)


C. Format Tags
These tags can fine tune the format of the graph.

\nolabels
Remove the axes labels, including the numbers indicated on the axes.

\nolabelnums
Just Remove the numbers indicated on the axes.

\noaxes
Remove the axes.

\nogrids
Remove the grids.

\sideaxes
Put the x-axis (and y-axis if no negative value) to the side(s) of the graph.


D. Text style Tags
These tags can set the style of the texts.

\it{text}
Make the text becomes italic.

^{text}
Make the text becomes superscript.

_{text}
Make the text becomes subscript.


E. Symbol Tags
Insert the symbols as shown in the pull-down menu. That's it.


F. Greek Letters
Click the "Greek keyboard" button to toggle the greek keyboard, and click again to hide it. The greek letter you clicked is inserted into the textarea.
